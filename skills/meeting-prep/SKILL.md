# Meeting Prep

When asked to prepare for a meeting, follow these steps in order:

## 1. Search the vault

- Search for the contact by name, company, and email
- Use hybrid search to find matching meeting notes, contact profiles, and project files
- Look for any prior meeting briefs, call notes, or references to the person

## 2. Review existing context

- Read any matching files found in the vault
- Extract key details: role, company, past conversations, shared interests, open threads
- Note any action items or follow-ups from previous meetings

## 3. Fill gaps with web research

- Only search the web for information NOT already in the vault
- Research the person's current role, company, and recent activity
- Check LinkedIn, company website, and recent news

## 4. Generate the meeting brief

Format the brief as:

```
# Meeting Brief: [Name] — [Date]

## Who
- Name, role, company
- How you're connected (if known from vault)

## Context
- Why you're meeting (from calendar or user input)
- Any prior interactions (from vault notes)

## Key Talking Points
- Relevant topics based on vault context and research
- Mutual interests or shared connections

## Open Threads
- Unresolved items from past conversations
- Follow-ups that were promised

## Quick Research
- Recent news or activity (only what's new beyond vault)
```

## 5. Save to vault

- Write the brief to: `0-Capture & Process/Meeting Prep - [Name] [Date].md`
- Use the format `YYYY-MM-DD` for the date

## 6. Return the contents of the brief

- Return the brief that you wrote to the user
